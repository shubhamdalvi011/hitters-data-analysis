Hitters Data Analysis

An Exploratory Data Analysis (EDA) project performed on the Hitters dataset using Python. This project focuses on cleaning, exploring, and visualizing baseball player statistics to uncover meaningful patterns and insights into player performance.

Project Overview

This project demonstrates the complete Exploratory Data Analysis (EDA) process, including:

- Data Loading
- Data Cleaning
- Missing Value Analysis
- Duplicate Detection
- Descriptive Statistics
- Performance Insights

Key Insights

Some insights obtained from the analysis include:

- people who have batted for more than 400 matches
- name of the player who has hit max home runs
- league wise total number of players
- division wise total amount of salary invested
- Five most senior players
